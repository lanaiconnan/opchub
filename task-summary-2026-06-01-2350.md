## 目标
修复OPC协作网络中"我的协作"页面删除按钮报404错误的问题。

## 问题分析
- **错误现象**：点击删除按钮时显示 `删除失败：Request failed with status code 404`
- **根本原因**：后端 `server.js` 缺少 `DELETE /opc/delete/:id` 路由处理
- **前端调用**：`MyCollaborations.jsx` 中的 `handleDelete` 函数正确调用了 `/opc/delete/:id` 接口

## 解决方案
在 `server.js` 中添加了 `DELETE /opc/delete/:id` 路由：
- 从 URL 参数中获取 OPC 的 `id`
- 在 `opcList` 数组中查找对应索引
- 如果找不到返回 404 错误
- 找到则使用 `splice()` 删除该元素
- 返回成功响应 `{ success: true, message: '删除成功' }`
